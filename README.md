#  Estudos em Java – Programação Orientada a Objetos

Repositório contendo diversos **exercícios práticos em Java**, desenvolvidos com foco no aprendizado dos **fundamentos da linguagem** e dos principais conceitos de **Programação Orientada a Objetos (POO)**.

Este projeto reúne exemplos simples e didáticos, ideais para quem está iniciando em Java ou revisando conceitos essenciais.


##  Contexto Acadêmico
Este repositório reúne os projetos desenvolvidos no 2º semestre da graduação, na disciplina de Técnicas de Programação do curso de Desenvolvimento de Software Multiplataforma da FATEC de Registro, demonstrando a aplicação prática dos conceitos estudados ao longo do semestre.


##  Conteúdos Abordados


- Estrutura básica de um programa Java (`main`)
- Entrada de dados com `Scanner`
- Operadores aritméticos
- Métodos com e sem parâmetros
- Construtores
- Encapsulamento (`private`, getters e setters)
- Classes e objetos
- Matrizes
- Laços de repetição (`for`)
- Organização de pacotes
- Padrão de projeto **Builder**
